# Liquid syntax error (line 26): Unknown tag 'url' ?

Django의 템플릿 문법이 Jekyll의 [liquid](https://jekyllrb.com/docs/liquid/)와 같아서 루비 문법으로 해석하다보니 충돌이 발생한다. 해결책은 아래와 같다:


{% raw %}{%{% endraw %} raw %}
{% raw %}```
<a href="{% url 'social:begin' 'oauth2-provider-name' %}">Login</a>
```{%{% endraw %} endraw %}
