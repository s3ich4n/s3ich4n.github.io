# OAuth2란?

OAuth2는 애플리케이션이 페이스북, Github, 등과 같은 HTTP 서비스를 통해 인증으로 제한된 유저 계정을 얻어오는 인증 프레임워크다. 이는 유저정보를 호스트하는 서비스에게 받거나, 서드파티 앱이 유저 계정에 접근할 수 있도록 유저 인증을 위임 받아 작동한다.
