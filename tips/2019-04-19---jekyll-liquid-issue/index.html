Django 코드를 Jekyll에 작성할 때 Liquid syntax error가 뜬다면?
tips
2019-04-19T12:00:00.000Z

Liquid syntax error (line 26): Unknown tag ‘url’ ?

Django의 템플릿 문법이 Jekyll의 liquid와 같아서 루비 문법으로 해석하다보니 충돌이 발생한다. 해결책은 아래와 같다:

{% raw %}{%{% endraw %} raw %} {% raw %}``` Login
